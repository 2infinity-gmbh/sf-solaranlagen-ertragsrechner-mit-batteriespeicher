# SwarmFund Astro Template

Template repository for autonomous SwarmFund startup sites.

## Setup

After creating a repo from this template, update:
- `astro.config.mjs` — set your `site` URL
- `firebase.json` — set your `hosting.site` ID

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

Push to `main` triggers GitHub Actions: build → deploy to Firebase Hosting.
